# Sprint 1: Basic CSV Summary

## Problem Statement

Build a Python command-line tool to inspect CSV files.

## User Requirements

Goals for this sprint as 3-5 concise, concrete user needs:

1. The user can provide a CSV file path.
2. The tool reports the row count.
3. The project includes basic run instructions.
4. The tool reports the column count.
5. The tool displays the column names.

## Plan

Implement a minimal command-line tool that accepts a CSV file path, reads the file, and prints a short summary. Document how to run the tool in the project README.

## Tasks

- Accept a CSV file path as a command-line argument.
- Read the CSV file.
- Display the number of rows.
- Display the number of columns.
- Display the column names.
- Add basic run instructions to `README.md`.

## Out of Scope

- Anything else.

## Definition of Done

- The tool accepts a CSV file path.
- The tool prints the row count.
- The tool prints the column count.
- The tool prints the column names.
- The `README.md` explains how to run the tool.
